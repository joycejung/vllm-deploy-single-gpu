# vllm-deploy-single-gpu
 基于 vLLM 的单机单卡大模型推理部署 Demo，完整复现高吞吐推理的核心流程，为后续多卡/分布式部署打下基础。
 ---
 ## 📌 项目背景
 随着大模型应用的普及，低延迟、高吞吐的推理部署能力成为工程落地的关键。
 本项目从最基础的单机单卡场景入手，深入理解 vLLM 的核心机制（如 PagedAttention、请求调度、显存管理），为后续扩展到多卡、多机分布式推理打好理论与实践基础。
 ---
 ## 🚀 功能特性
 - ✅ 单机单卡环境下，基于 vLLM 实现大模型推理服务
 - ✅ 支持 OpenAI 兼容 API 接口调用（方便前后端集成、压测）
 - ✅ 实现基础请求调度与 KV 缓存管理逻辑
 - ✅ 显存占用优化（FP16/FP8 混合精度、动态批处理）
 - ✅ 提供简单的性能压测脚本，可直观对比不同参数的吞吐/延迟表现
 ---
